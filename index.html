There are 5 primary classes in bounce.css listed below

To get style in your HTML file just add these classes to the respective elements
example: to get cyan button add the class "cyan" to the button

This method is universal to the bounce.css, every element having "cyan" class will follow the above property.
However if you don't want any background color on your elements, do not add any class to your elements
